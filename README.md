# Reki - blazingly fast and aesthetic git log viewer

**Reki** is a blazingly fast, aesthetic git log viewer written in Rust.(Working In Progress - aiming to be the most beautiful way to read history)

Inspired by [yazi](https://github.com/sxyazi/yazi), [tig](https://github.com/jonas/tig), and [lazygit](https://github.com/jesseduffield/lazygit), Reki aims to provide a visually appealing and efficient way to explore git commit history directly from the terminal.
